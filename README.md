Hola!!, Mini-Blog es un Proyecto en el cual la idea fue armar una app tipo blog, con el que un Usuario pueda Crear Entradas, Modificarlas, Publicarlas.

La idea detras del Proyecto fue aplicar los conocimientos a un Proyecto Practico, en el cual pudiera Utiliazar lo Aprendido en Js,HTML 5, Programacion en general 
Y Tambien Desafiarme a Poder Resolver Problematicas que fueran Surgiendo al tener que aprender herramientas que no conocia o habia utilizado muy poco.

En este Proyecto pude Utilizar Node y Express, Herramientas que habia Utilizado poco y nada, Ademas de Sumarle ejs.Tambien Trabaje con Bcrypt,Zod y Express-session
Herramientas Nuevas, pero que me parecen que suman mucho para este proyecto como para futuros.

Lo que mas me gusto aprender es el circuito de back-end. Express, Rutas, Registro-Login, Verificaciones, etc y Tambien poder juntar por primera vez en un solo proyecto front y back, y experimentar como implementar la interaccion entre las dos partes.
